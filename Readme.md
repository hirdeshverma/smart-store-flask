# Smart Store Flask

This is a simple Flask and Python-based inventory management web application.

It allows users to add, view, edit, and delete products while storing data in a JSON file. This project was built to practice Flask fundamentals, CRUD operations, form validation, and Git/GitHub.


## Features

- Add new products
- View all products
- Edit existing products
- Delete products
- Product name validation
- Duplicate product prevention
- Price validation
- Data stored in a JSON file

## Technologies Used
- Python
- Flask
- HTML
- JSON
- Git
- GitHub
